# Student And Admin Workbench UI Design

## Goal
- 优化学生首页和管理员看板的排列与显示，让两个入口页和学生选座页一样偏“工作台”而不是普通卡片堆叠。

## Scope
- 学生首页：突出今日状态、快捷入口、当前预约、锁位权益和时间线。
- 管理员看板：突出日期控制、核心运营指标、异常处理、锁位运维和区域利用率。
- 全局样式：在浅色科技风基础上增加轻量动效，不引入深色大屏风或复杂动画。

## Layout
- 学生首页使用两列工作台布局：主列展示欢迎区、当前预约、今日时间线；侧列展示规则/锁位权益、常用区域、最近记录。窄屏改为单列，操作按钮靠前。
- 管理员看板使用顶部控制区、指标矩阵、两列运营区。异常指标和释放过期锁位操作靠前，区域利用率占主列。
- 卡片保持 8px 圆角以内，使用浅色渐变、细边框、柔和 hover 和进入动画增强科技感。

## Verification
- 更新 App 级测试，确认学生首页出现 `学生预约工作台`、`今日状态`、`快捷入口` 等结构；管理员看板出现 `管理员运营工作台`、`运行概览`、`异常处理`。
